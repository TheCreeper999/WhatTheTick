What the tick est un mod ajoutant une commande (/tick) qui permet de modifier le nombre de ticks par seconde du jeu !
Pour cela, il faut exécuter la commande comme ceci :
/tick <nb>
D'où <nb>, le nombre de ticks. Par défaut, minecraft est à 20 ticks par seconde.

Téléchargement : http://forum.ironcraft.fr/files/file/49-what-the-tick/
